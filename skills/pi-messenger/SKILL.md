# Pi Messenger

Use for coordinating multiple agents inside one project.

- Run `/messenger` for the crew/task management UI.
- Use `pi_messenger()` for direct automation: `join`, `plan`, `work`, `task.*`, `status`.
- Useful for larger coordinated implementation batches.

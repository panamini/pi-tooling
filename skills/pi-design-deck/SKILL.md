# Pi Design Deck

Use when you need a visual side-by-side option deck to compare architecture or UI choices.

- Start with the `design_deck` tool.
- Open a saved deck, add generated options, and keep selecting preferred paths.
- Useful before implementation planning or UI/architecture decisions.

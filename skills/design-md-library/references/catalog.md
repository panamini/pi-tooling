## Notes

- Dotted folder names such as `linear.app`, `mistral.ai`, `opencode.ai`, `together.ai`, and `x.ai` are valid routing targets.
- Each pack usually contains `DESIGN.md`, `README.md`, `preview.html`, and `preview-dark.html`.
- Prefer `DESIGN.md` for agent context. Use preview files only when you need extra visual confirmation.
